Data orgainization

Species196-L and Species196-U Datasets

For the Species196-L classification dataset, images will be downloaded and organized in the ImageNet style using the provided download script. Meanwhile, for the Species196-L object detection dataset, images will be downloaded and arranged into train and val subfolders following the COCO format, also using the provided download script.

For Species196-U, 4 separate JSON files for Insecta, Mollusca, Weeds, and Insects' other life cycles will be provided. We suggest using img2dataset to download these unlabeled images. Please note that image links might expire, so we've included some redundancy in the dataset.
